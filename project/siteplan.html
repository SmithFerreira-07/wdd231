Site Name:

Surf Buddy

Domain: surfbuddy.org

This name represents a helpful companion for surfers looking to find the perfect conditions and improve their surfing experience

Site Purpose

This site provides surfers with essential tools for tracking wave conditions, weather forecasts, and surf spots. Using APIs like Stormglass.io and OpenWeather, Surf Buddy helps surfers make informed decisions about when and where to catch the best waves.

Scenarios

  • What are the wave conditions at my favorite beach today?
  • Is the weather suitable for surfing this weekend?
  • Which nearby surf spots have the best conditions right now?
  • What's the tide schedule for the next week?
  • Are there any surfing events happening in my area?

Typography
